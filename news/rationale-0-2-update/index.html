Rationale 0.2 Update

With Rationale's new multi-option analysis feature, you'll have the tools you need to tackle even the toughest multi-option problems. Try out the new multi-option analysis feature in Rationale 0.2 today!

Engineering Group
January 09, 2023 • 4 minutes read

Rationale is a decision-making AI powered by the latest version of GPT-3 and in-context learning. It assists business owners, managers, and individuals in making tough decisions by generating a pros and cons list, SWOT analysis, and multi-option analysis (newly available in 0.2!). By considering all relevant factors, Rationale helps users make rational decisions.

Multi-option analysis

Making tough decisions can be difficult, especially when there are multiple options to choose from. This type of problem, known as a multi-option problem, can be especially challenging for humans because we have a tendency to overthink and weigh the pros and cons of each option.

That's where Rationale comes in. Our AI-powered decision-making tool, equipped with the latest version of GPT-3 and in-context learning, can help you navigate these situations by generating a comprehensive analysis of all your options. Simply enter all of your indecisive options (can be more than two!) into Rationale and our tool will analyze them based on four common criteria or perspectives. Then, Rationale will provide you with the most rational choice among all options, giving you the confidence and clarity you need to move forward.

With Rationale's multi-option analysis feature, you'll have the tools you need to tackle even the toughest multi-option problems. Try out the new multi-option analysis feature in Rationale 0.2 today!

4 new achievements to unlock

To accompany the new multi-option mode, Rationale 0.2 also introduces four new achievements for you to unlock. These achievements are designed to encourage you to try out the new multi-option analysis feature and see all that it has to offer.

The first new achievement is the "Multi-Choice Master" award, which you can unlock by using the multi-option analysis feature just one time. The next achievement is the "Multi-Choice Maniac" award, which you can unlock by using the multi-option analysis feature 10 times.

If you're feeling particularly ambitious, you might want to try for the "Option Overlord" award, which you can unlock by entering and analyzing more than two options at a time in the multi-option mode. And for the ultimate challenge, there's the "Option Overflow" award, which you can unlock by entering and analyzing more than five options at a time in the multi-option mode.

These new achievements offer a fun and rewarding way to explore the multi-option analysis feature and see all that it has to offer. Give them a try and see how many you can unlock!

Faster response

In addition to the new multi-option analysis feature, we're excited to announce that Rationale 0.2 also boasts faster response speed and improved robustness. On average, the response time for Pros & Cons and SWOT mode is 2x faster (up to 3x!) than the previous version, making it easier and quicker to get the insights you need.

But faster response time isn't the only improvement we've made. The response in Rationale 0.2 is also more robust, meaning it can handle a wider range of input and provide more accurate and relevant results.

Bug fixes on delete thread and swipe to give feedback

In addition to the new features and improvements mentioned above, Rationale 0.2 also includes a number of bug fixes to ensure a smoother user experience.

One bug that we've addressed is an issue with the delete thread feature. In previous versions of Rationale, some users experienced problems when trying to delete an analysis from the thread view. We're happy to report that this issue has been fixed in Rationale 0.2, so you can easily delete your analyses as needed.

We've also fixed an issue with the swipe to give feedback feature. Some users encountered problems when attempting to provide relevancy feedback by swiping right or left on items in the analysis card. This issue has been resolved in Rationale 0.2, so you can easily provide feedback and help us improve the app.

Overall, these bug fixes help ensure you have a seamless and hassle-free experience when using Rationale. We're always working to make our app as user-friendly and reliable as possible, and these bug fixes are just one example of our commitment to your satisfaction.
