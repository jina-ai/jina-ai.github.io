Announcing Jina 0.7.0

Alex C-G
21 October, 2020

We are excited to release Jina v0.7.0! Jina is an easier way to do a neural search on the cloud.

Highlights of this release include:

  • Flow evaluation support
  • Support for preventing duplicates Documents in the index
  • Flow visualization support

In this post we'll highlight some of the key new features. For full release notes, including bug fixes, breaking changes, and sample code, please check our GitHub repo

‚¨ÜÔ∏è Major Features and Improvements

Completeness

  • Evaluation is fully supported by Jina. jina.executors.evaluators and jina.drivers.evaluate have been introduced to make this happen. Now you can use different metrics to evaluate the Flow. No matter whether you want to evaluate the whole Flow or just part of it, the evaluation can be done smoothly without stopping the running Flow.

  • To prevent duplicates in the index, UniquePbIndexer and UniqueVectorIndexer are introduced together with the corresponding drivers in jina.drivers.cache. Please refer to docs.jina.ai for more details.

Usability

  • Add visualization for Flow. Calling plot() function of Flow gives a better view of how the Flow looks.
