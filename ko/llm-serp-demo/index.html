비율 제한
속도 제한은 두 가지 방법, 즉 RPM(분당 요청 수)과 TPM(분당 단어 수)으로 추적됩니다. 제한은 IP/API 키별로 적용되며 먼저 도달한 임계값(RPM 또는 TPM)에 따라 제한에 도달할 수 있습니다. 요청에 API 키가 제공되는 경우 속도 제한은 IP 주소가 아닌 키별로 추적됩니다.
