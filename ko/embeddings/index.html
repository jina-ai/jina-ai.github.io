jina-clip-v1을 사용할 때 토큰을 어떻게 계산하나요?
keyboard_arrow_down
토큰은 텍스트 길이와 이미지 크기를 기준으로 계산됩니다. 요청 텍스트의 경우 토큰은 표준 방식으로 계산됩니다. 요청의 이미지에 대해 다음 단계를 수행하세요. 1. 타일 크기: 각 이미지는 224x224 픽셀 크기의 타일로 나뉩니다. 2. 적용 범위: 입력 이미지를 완전히 덮는 데 필요한 타일 수를 계산합니다. 이미지 크기가 224로 정확히 나누어지지 않더라도 부분 타일을 전체 타일로 계산합니다. 3. 총 타일 수: 이미지를 덮는 총 타일 수에 따라 비용이 결정됩니다. 예를 들어 이미지가 500x500픽셀인 경우 3x3 타일로 덮여 결과적으로 9개의 타일이 됩니다. 4. 비용 계산: 각 타일은 이미지 처리의 최종 비용에 영향을 미칩니다. 각 타일의 가격은 1000토큰입니다. 예: 500x500픽셀 크기의 이미지의 경우: • 이미지는 224x224 픽셀의 타일로 나누어집니다. • 필요한 총 타일 수는 3(가로) x 3(세로) = 9개의 타일입니다. • 비용은 9*1000 = 9000 토큰입니다.
