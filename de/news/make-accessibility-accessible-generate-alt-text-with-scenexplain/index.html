Barrierefreiheit zugänglich machen: Alt-Text mit der SceneXplain API erstellen

SceneXplain ist Ihr Verbündeter für Barrierefreiheit, der es einfach macht, Alt-Texte für Bilder zu generieren, um sehbehinderte Nutzer zu unterstützen und SEO zu verbessern
Alex C-G • 4 Minuten gelesen

Barrierefreiheit (oder kurz "a11y") entwickelt sich schnell zu einem wichtigen Bestandteil der Webentwicklung und des E-Commerce. Früher wurden Hilfsmittel für Barrierefreiheit wie Alt-Text oder farbenblinden-freundliche Farbschemata von Entwicklern und Unternehmen nicht als hohe Priorität angesehen. Doch heute, mit Gesetzgebungen zur Barrierefreiheit aus Europa und den USA, ist es wichtiger denn je, Ihre Website barrierefrei zu gestalten.

💡
Alt-Text oder alternativer Text ist eine kurze Beschreibung eines Bildes, die auf Websites und in digitalen Dokumenten verwendet wird. Er hilft Menschen, die das Bild nicht sehen können, zu verstehen, worum es geht. Dazu gehören Menschen, die aufgrund von Sehbehinderungen Screenreader nutzen, und solche mit langsamen Internetverbindungen, bei denen Bilder nicht geladen werden. Alt-Text ist auch für Suchmaschinen nützlich, um Bilder zu verstehen und zu indexieren.

Aber wie können Sie für jedes Bild auf Ihrer Website Alt-Text erstellen? Manuell jedes Bild durchzugehen und Alt-Text zu schreiben könnte ewig dauern, besonders wenn Sie Tausende (oder Millionen) von Bildern haben. Und wenn täglich neue hinzukommen, wird es zu einem endlosen Kampf.

Hier kommt SceneXplain ins Spiel. Es ist Ihr a11y Verbündeter! Sie können einfach ein Bild hochladen und dafür Alt-Text erhalten, ohne sich selbst den Kopf über die Formulierung zerbrechen zu müssen.

Wenn Sie, sagen wir, ein paar Dutzend Bilder haben, ist dies eine gute Möglichkeit, Ihrem Gehirn eine Pause zu gönnen. Aber Sie müssen immer noch selbst das ganze Klicken und Ziehen erledigen. Ihr Gehirn gewinnt, aber Ihre Finger nicht. Und wenn Sie ein paar tausend Bilder haben? Vereinbaren Sie gleich einen Termin beim Arzt für Ihr Karpaltunnelsyndrom.

Wenn es nur einen Weg gäbe, das Ganze zu automatisieren. Dann könnten sich Ihr Gehirn und Ihre Finger beide auf interessantere Dinge konzentrieren.

Hier kommt die SceneXplain API ins Spiel. Sie können ein Skript schreiben, das Ihre tausenden von Bildern durchgeht, sie stapelweise an SceneXplain sendet und eine CSV-Datei mit den Ergebnissen generiert (oder mit etwas mehr Programmierung direkt in Ihren Workflow integriert.)

Schließlich weiß man ja, was sie sagen. Man kann Glück nicht ohne API buchstabieren.

Kurz gesagt, Sie können ein Python-Programm (oder eine andere Sprache) schreiben, das über die API mit SceneXplain kommuniziert und Ihren gesamten Alt-Tagging-Prozess automatisiert. Wir haben ein Python-Snippet, das genau das tut.

💡
Können Sie den Code nicht auf Ihrem eigenen Computer ausführen? Lesen Sie weiter bis zum Google Colab Notebook, das Ihnen die Nutzung im Browser ermöglicht.

So verwenden Sie es:

  1. Installieren Sie die requests Bibliothek:
pip install requests
  1. Gehen Sie zur API-Seite von SceneXplain, um einen geheimen Schlüssel zu generieren und kopieren Sie ihn in die Zwischenablage.
  2. Fügen Sie ihn in den untenstehenden Python-Code ein.
  3. Kopieren Sie eine Bild-URL in den Code, wo .... steht.
  4. Führen Sie den Code aus!
import requests
import json

# generate token on SceneXplain's API page
YOUR_GENERATED_SECRET = "your_generated_secret_here"

data = {
  "data": [
    {
      "task_id": "alt_text",
      "languages": [
        "en"
      ],
      "image": "..." # change to image URL
    }
  ]
}

headers = {
  "x-api-key": f"token {YOUR_GENERATED_SECRET}",
  "content-type": "application/json",
}

response = requests.post("https://api.scenex.jina.ai/v1/describe", headers=headers, json=data)
print(response.json())

Das Notebook geht über das einfache Python-Snippet oben hinaus. Es lädt auch einen Beispieldatensatz herunter und exportiert die Ergebnisse in eine CSV-Datei.

tagÜber das Notebook hinaus: Die API im echten Leben nutzen

Natürlich sind Sie bei der Verwendung der SceneXplain API nicht auf Python beschränkt. Jede Sprache mit einer HTTP-Bibliothek sollte gut funktionieren.

Hier ist dasselbe Code-Snippet von oben, diesmal in JavaScript:

const body = {
  "data": [
    {
      "task_id": "alt_text",
      "languages": [
        "en"
      ],
      "image": "..."
    }
  ]
};

const YOUR_GENERATED_SECRET = 'your_generated_secret_here';

fetch('https://api.scenex.jina.ai/v1/describe', {
  headers: {
    'x-api-key': `token ${YOUR_GENERATED_SECRET}`,
    'content-type': 'application/json'
  },
  body: JSON.stringify(body),
  method: 'POST'
}).then(async (resp) => {
  if (resp.ok) {
    const data = await resp.json();
    console.log(data);
  }
});

Und hier als cURL Befehl:

curl "https://api.scenex.jina.ai/v1/describe" \
  -H "x-api-key: token $YOUR_GENERATED_SECRET" \
  -H "content-type: application/json" \
  --data '{
  "data": [
    {
      "task_id": "alt_text",
      "languages": [
        "en"
      ],
      "image": "..."
    }
  ]
}'
