FAQ

question_answer
Qu'est-ce que SERP ?
keyboard_arrow_down
SERP signifie Search Engine Results Page, c'est-à-dire la page affichée par les moteurs de recherche en réponse à une requête d'un utilisateur. Elle comprend une liste de résultats, souvent accompagnée d'informations supplémentaires telles que des URL, des extraits de code, un nom de domaine.
question_answer
Donc tous les résultats de recherche sont générés par LLM ?
keyboard_arrow_down
Oui, LLM-as-SERP utilise de grands modèles linguistiques pour générer des résultats de recherche.
question_answer
Pourquoi certaines URL dans les résultats de recherche sont-elles cassées et mènent-elles à des pages 404 ?
keyboard_arrow_down
Hallucinations.
question_answer
Alors pourquoi certaines URL sont-elles réelles et mènent-elles à la page réelle ?
keyboard_arrow_down
Ils peuvent être dans les données de formation du modèle.
question_answer
Puis-je l'appeler via l'API ?
keyboard_arrow_down
Oui, vous pouvez utiliser l'API LLM-as-SERP pour générer des résultats de recherche. Cliquez sur le bouton ci-dessous.

Limite de taux
Les limites de débit sont suivies de deux manières : RPM (requêtes par minute) et TPM (jetons par minute). Les limites sont appliquées par IP/clé API et peuvent être atteintes en fonction du seuil (RPM ou TPM) atteint en premier. Notez que lorsque la clé API est fournie dans la demande, les limites de débit sont suivies par clé et non par adresse IP.
