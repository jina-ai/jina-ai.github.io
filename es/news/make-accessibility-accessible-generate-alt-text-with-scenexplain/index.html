Haciendo la Accesibilidad Accesible: Crea Texto Alternativo con la API de SceneXplain

SceneXplain es tu aliado en accesibilidad, facilitando la generación de textos alternativos para imágenes que ayudan a usuarios con discapacidad visual y mejoran el SEO
Alex C-G • 4 minutos de lectura

La accesibilidad (o "a11y" en forma abreviada) se está convirtiendo rápidamente en una parte importante del desarrollo web y el comercio electrónico. En el pasado, las ayudas de accesibilidad como el texto alternativo o los esquemas de colores adaptados para daltónicos no eran considerados prioritarios por los desarrolladores y las empresas. Pero ahora, con la legislación sobre accesibilidad de Europa y Estados Unidos, hacer que tu sitio web sea accesible es más importante que nunca.

💡
El texto alternativo, o texto alt, es una breve descripción de una imagen utilizada en sitios web y documentos digitales. Ayuda a las personas que no pueden ver la imagen a entender de qué se trata. Esto incluye a personas que usan lectores de pantalla debido a discapacidades visuales y aquellos con conexiones lentas a internet donde las imágenes no se cargan. El texto alternativo también es útil para que los motores de búsqueda entiendan e indexen las imágenes.

Pero, ¿cómo puedes crear texto alternativo para cada imagen en tu sitio web? Revisar manualmente cada imagen y escribir texto alternativo podría llevar una eternidad, especialmente si tienes miles (o millones) de imágenes. Y si se están agregando más cada día, se convierte en una batalla sin fin.

Ahí es donde entra SceneXplain. ¡Es tu aliado de a11y! Puedes simplemente subir una imagen y obtener texto alternativo para ella sin tener que romperte la cabeza pensando en la redacción tú mismo.

Si tienes, digamos, algunas docenas de imágenes, esta es una buena manera de darle un descanso a tu cerebro. Pero aún necesitas hacer todos los clics y arrastres tú mismo. Tu cerebro gana, pero tus dedos no. ¿Y si tienes miles de imágenes? Llama al doctor ahora para reservar tu cita por síndrome del túnel carpiano.

Si tan solo hubiera una forma de automatizar todo el proceso. Entonces tu cerebro y tus dedos podrían ambos concentrarse en cosas más interesantes.

Ahí es donde entra la API de SceneXplain. Puedes escribir un script que recorra tus miles de imágenes, las envíe en lotes a SceneXplain, y genere un archivo CSV con los resultados (o con un poco más de código, integrarlo directamente en tu flujo de trabajo).

Después de todo, ya sabes lo que dicen. No puedes deletrear felicidad sin APIcidad.

En resumen, puedes escribir un programa en Python (o cualquier otro lenguaje) para comunicarte con SceneXplain a través de su API y automatizar todo tu proceso de etiquetado alternativo. Tenemos un fragmento de Python que hace exactamente eso.

💡
¿No puedes ejecutar el código en tu propia computadora? Sigue leyendo hasta el notebook de Google Colab que te permite usarlo en tu navegador.

Así es como se usa:

  1. Instala la biblioteca requests:
pip install requests
  1. Ve a la página de API de SceneXplain para generar una clave secreta y cópiala al portapapeles.
  2. Pégala en el código Python a continuación.
  3. Copia una URL de imagen en el código donde dice .....
  4. ¡Ejecuta el código!
import requests
import json

# generate token on SceneXplain's API page
YOUR_GENERATED_SECRET = "your_generated_secret_here"

data = {
  "data": [
    {
      "task_id": "alt_text",
      "languages": [
        "en"
      ],
      "image": "..." # change to image URL
    }
  ]
}

headers = {
  "x-api-key": f"token {YOUR_GENERATED_SECRET}",
  "content-type": "application/json",
}

response = requests.post("https://api.scenex.jina.ai/v1/describe", headers=headers, json=data)
print(response.json())

tagMás allá del notebook: Usando la API en la vida real

Por supuesto, no estás limitado a Python cuando usas la API de SceneXplain. Cualquier lenguaje que tenga una biblioteca HTTP debería funcionar bien.

Aquí está ese mismo fragmento de código de arriba, esta vez en JavaScript:

const body = {
  "data": [
    {
      "task_id": "alt_text",
      "languages": [
        "en"
      ],
      "image": "..."
    }
  ]
};

const YOUR_GENERATED_SECRET = 'your_generated_secret_here';

fetch('https://api.scenex.jina.ai/v1/describe', {
  headers: {
    'x-api-key': `token ${YOUR_GENERATED_SECRET}`,
    'content-type': 'application/json'
  },
  body: JSON.stringify(body),
  method: 'POST'
}).then(async (resp) => {
  if (resp.ok) {
    const data = await resp.json();
    console.log(data);
  }
});

Y esta vez como un comando cURL:

curl "https://api.scenex.jina.ai/v1/describe" \
  -H "x-api-key: token $YOUR_GENERATED_SECRET" \
  -H "content-type: application/json" \
  --data '{
  "data": [
    {
      "task_id": "alt_text",
      "languages": [
        "en"
      ],
      "image": "..."
    }
  ]
}'
